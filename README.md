# weather-pwa
Weather Progressive Web App. Inspired by https://github.com/udacity/ud811 (1-22).

Vue.js progressive webapp to checkout the weather in your locations.

Using open apis:
> freegeoip.net
> openweathermap.org
> google places apis

Languages:
> HTML
> CSS
> Javascript / JQuery

Technologies Involved:
> Javascript Promises
> Vue.js
> IndexedDb
> Service Workers
